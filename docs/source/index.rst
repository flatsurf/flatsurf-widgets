
flatsurf_widgets
=====================================

Version: |release|

Jupyter Widgets for flatsurf


Quickstart
----------

To get started with flatsurf_widgets, install with pip::

    pip install flatsurf_widgets

or with conda::

    conda install flatsurf_widgets
